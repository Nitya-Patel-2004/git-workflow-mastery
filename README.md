
Git Workflow Mastery Project

Description:
Developed a hands‑on project showcasing professional Git practices across the full development lifecycle.

Goal2 : Create a merge conflict and resolve it manually before merging a PR on GitHub.

Key Achievements:
- Implemented branching and merging workflows to safely integrate new features.
- Simulated collaboration with Pull Requests, resolving merge conflicts manually.
- Maintained a clean commit history using interactive rebase (squash/fixup).
- Demonstrated safe undoing techniques with reset and revert for local and remote changes.
- Applied versioning best practices by creating lightweight and annotated Git tags for releases.
  Outcome:
  Repository reflects industry‑standard Git usage, highlighting readiness for collaborative software development and disciplined version control.
